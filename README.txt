

The folder contains the source code Cachesim. It also has runnable jar file and a script file (start.sh) to run the jar.

Trace files are also included in CS203Lab2.

Go to command prompt and cd into the folder CS203Lab2.

Example run:

./start.sh -i Cachesim/gcc-10K.memtrace -cs 524288 -bs 16 -w 0

OR

./start.sh -i Cachesim/gcc-10K.memtrace -cs 524288 -bs 16 -w 0 -vs 4
